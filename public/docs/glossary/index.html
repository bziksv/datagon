Глоссарий

Краткие определения сущностей и процессов, которые встречаются в панели и в REST API.

Конкуренты и парсинг​

ТерминЧто это
Проект (конкурент)Запись в разделе Конкуренты: сайт конкурента + CSS-селекторы для вытягивания цены, названия, SKU и т.д. с HTML-страницы карточки товара.
Очередь (pages)Список URL, которые нужно скачать и разобрать парсером. Статусы: ожидание, успех, ошибка и др. Экран Очередь.
ПарсингHTTP-запрос к странице конкурента + разбор HTML по селекторам + запись строки в prices (результаты).
Результаты (prices)Таблица собранных цен/наличия по конкурентам. Экран Результаты.

Наш каталог​

ТерминЧто это
Мой сайт (источник)Подключение к вашей внешней БД (Bitrix, Webasyst и т.п.). Экран Мои сайты.
my_productsЛокальная таблица в БД Datagon: «наши» товары, подтянутые из источников. Экран Мои товары.
is_activeПризнак того, что позиция есть в последнем синке источника. Перед синком записи помечаются неактивными; найденные в источнике снова становятся активными.
source_enabledПризнак на стороне источника (включён/выключен товар в вашей витрине) — используется в фильтрах.

Сопоставление и МойСклад​

ТерминЧто это
Сопоставление (матчинг)Поиск пар «наш товар ↔ товар конкурента» по SKU и/или по названию с оценкой похожести. Экран Сопоставление.
Кандидат / pendingНайденная пара, ожидающая ручного подтверждения или отклонения.
ms_exportЛокальная таблица выгрузки номенклатуры из API МойСклад. Экран МойСклад.
Связь с МСВ «Моих товарах»: есть ли соответствие кода товара строке в ms_export (нормализация при синке).

Система​

ТерминЧто это
Логи / процессыСводка фоновых задач: глобальный синк, МойСклад, парсинг, матчинг. Экран Логи.
НастройкиГлобальные параметры парсинга, синхронизации, токены, пользователи. Настройки.
Веб-справкаЭтот сайт на Docusaurus: исходники docs-docusaurus/docs/, сборка в public/docs/, URL /docs/.
Воркер (worker.js)Отдельный процесс (или сервис), который забирает pages в обработку и пишет в prices; без него очередь «не двигается».
DiscoverАвтодобавление URL в очередь по sitemap/правилам; лимиты discover_* в настройках.
dg_session / cookieСерверная сессия для доступа к /docs/* и панели; см. session-bridge.html в статике доков.
Нормализация SKUПриведение артикула к каноническому виду (пробелы, регистр) при сравнении my_products и ms_export / конкурента — логика на сервере.
Порог матчингаЧисло threshold: минимальная «уверенность» пары наш–чужой.

Если термин в интерфейсе не совпадает с таблицей — ориентируйтесь на подсказки (иконки «?») и на названия колонок в таблицах.
