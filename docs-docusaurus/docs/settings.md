---

Глобальные параметры приложения: парсер, синхронизация, сессии, пользователи.

## Основные группы

- пользователи и пароли;
- лимиты/паузы парсинга;
- параметры синхронизации `my_products`;
- параметры МойСклад;
- расписание автозапуска.

Запись параметров: `POST /api/settings`.
