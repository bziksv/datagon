Сводка фоновых задач Datagon.

## Раздел показывает

- статус глобальной синхронизации;
- состояние синхронизации МойСклад;
- автообход URL и очередь парсинга;
- статус сопоставления по выбранному сайту.

## Когда использовать

Если задача «зависла» или выполняется слишком долго — сначала проверяйте этот экран.
