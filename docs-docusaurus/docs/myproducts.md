---

Работа с локальным каталогом `my_products`: фильтры, сравнение с конкурентами, действия по цене.

## Возможности

- умный поиск и фильтры по сайту/статусу/связи с МС;
- сортировка по столбцам и через панель фильтра;
- сравнение с ценами конкурентов (ДМ/МК);
- точечное обновление строки и синхронизация цены.

## Сортировка

- по умолчанию: `ID с сайта` по возрастанию;
- клик по заголовку столбца меняет направление;
- выбранная сортировка сохраняется отдельно по пользователю.
