# Документация в репозитории

- **Веб-справка** (панель, REST API, деплой, архив плана UI): исходники в **`docs-docusaurus/docs/`**, сборка **`npm run docs:docusaurus:build`** из корня репозитория → статика в **`public/docs/`**, в приложении открывается как **`/docs/`**.

- **Nginx:** примеры конфигов в **`deploy/`** в корне репозитория (не входят в Docusaurus).

Обзор продукта, FAQ и ссылки на справку — в **`README.md`** в корне репозитория.
