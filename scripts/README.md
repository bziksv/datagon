# Скрипты репозитория

| Папка | Назначение |
|-------|------------|
| `docs/` | Документация: снимки экранов для `public/docs/` |
| `build/` | Сборка фронта в `public/architectui-react-pro/` |
| `qa/` | Smoke-проверки Playwright |

Команды из корня: `npm run docs:capture-screenshots`, `npm run build:datagon-spa`, `npm run test:datagon-smoke-e2e`.
